# Vacation App
## Usage
1. install NodeJS
https://nodejs.org/en/

2. install dependencies
```
 npm install
```

3. build application or
```
npm run build
```

4. quick start application
```
npm start

```
5. for clean .tmp files
```
npm run prebuild
```
6. to start application in static ip add parameter { --host yo.ur.i.p } in package.json script "start"
